Similization Ruleset Editor

The Similization Ruleset editor is an application that allows you to create and edit the rules behind the Similization game. The Ruleset editor allows you to control many aspects of the game, including:

  • Civilizations
  • Units
  • Resources
  • Technologies
  • Terrains
  • Improvements
  • Eras
  • Governments
  • Images

The Ruleset editor also comes with a Map editor, which allows you to create new maps to use during the game.
